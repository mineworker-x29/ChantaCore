# v0.33.8 CLI Agent Run Surface

Parent track: Internal General Agent Runtime MVP.

v0.33.8 builds on v0.33.7 Runtime OCEL Trace Emitter, v0.33.6 Agent Step Runner MVP, and v0.33.5 Safe Workspace Inspection Tool Pack. It exposes bounded CLI-facing command handlers as pure Python artifacts.

## Definition

The CLI Agent Run Surface parses CLI-style `argv` lists into `CLIAgentInvocation` artifacts, evaluates them against `CLIAgentSurfacePolicy`, dispatches only approved bounded internal commands, and returns `CLIAgentRunOutput` data.

CLI surface is not shell execution. CLI argument parsing is not OS command execution. CLI dispatch is not provider invocation. CLI output rendering is not file persistence.

## Safe Command Set

Supported command kinds are:

- `agent help`
- `agent status`
- `agent profile-preview`
- `agent prompt-preview`
- `agent session-preview`
- `agent inspect-path --path <path>`
- `agent inspect-tree --path <path>`
- `agent read-text --path <path>`
- `agent search --path <path> --query <query>`
- `agent reference-summary --path <path>`
- `agent step --supplied-output <json/text>`
- `agent step --mock-output <json/text>`
- `agent trace-preview --from-step-output`
- `agent trace-preview --from-session-snapshot`
- `agent no-op`

Unknown commands, shell-like arguments, provider/network/write/patch/install/reference-execution requests, and unsafe commands are denied or blocked as safe outcomes.

## Bridges

CLI inspect commands route only through v0.33.5 safe workspace inspection policy. CLI inspect is not arbitrary file read. Secret-like files remain skipped or denied by v0.33.5 policy.

CLI step commands route only through v0.33.6 bounded agent step runner with supplied/mock model output. A bounded CLI step is not autonomous agent runtime and does not call real providers.

CLI trace preview routes only through v0.33.7 returned trace packet creation. CLI trace preview is not persistent trace write, database write, runtime log write, or external trace sink usage.

## Reference Corpus Policy

`references/OpenCode` and `references/Hermes` may be inspected only through v0.33.5 bounded safe read-only path policy. v0.33.8 does not execute OpenCode, Hermes, or OpenClaw; does not import reference code; does not install dependencies; and does not run reference tests or package scripts.

## Output Policy

CLI output is bounded and redacted. It is returned as an in-memory string/data artifact. v0.33.8 does not write output files, logs, trace files, or database records.

## Flow

`argv list / CLI surface policy -> CLIAgentInvocation -> CLIAgentInvocationDecision -> optional v0.33.5/v0.33.6/v0.33.7 bounded call -> CLIAgentRunOutput -> CLIAgentRunReport -> V0338ReadinessReport -> v0.33.9 Consolidation`

## Hard Prohibitions

v0.33.8 does not execute shell commands, does not use subprocess, and never passes CLI argv to shell/subprocess.

v0.33.8 does not call real model providers, invoke provider SDKs, access OpenAI/Claude/local provider APIs, perform network calls, access credentials, execute general tools, or enable autonomous general agent runtime.

v0.33.8 does not write/edit/patch files, create/delete/rename/chmod/chown files, persist trace files, create database records, write runtime logs, use external trace sinks, implement UI runtime, mutate registries, mutate memory, perform external control, or grant authority.

v0.33.8 does not execute OpenCode/Hermes/OpenClaw, import reference code, install dependencies, or read secret-like files.

## Readiness

`ready_for_bounded_cli_agent_run` may be true for this bounded CLI surface.

`ready_for_execution`, `ready_for_shell_execution`, `ready_for_subprocess_execution`, `ready_for_real_model_invocation`, `ready_for_provider_invocation`, `ready_for_general_agent_execution`, `ready_for_general_tool_execution`, `ready_for_command_execution`, `ready_for_network_access`, `ready_for_credential_access`, `ready_for_workspace_write`, `ready_for_code_edit`, `ready_for_patch_application`, `ready_for_persistent_trace_write`, `ready_for_external_trace_sink`, and `ready_for_ui_runtime` remain false.

## Acceptance Criteria

- Safe command specs and pure parser/handler functions exist.
- Unknown and unsafe commands are blocked or denied.
- Safe inspect commands route only through v0.33.5.
- Step commands route only through v0.33.6 supplied/mock output.
- Trace previews route only through v0.33.7 returned trace packet creation.
- Output remains bounded, redacted, and non-persistent.
- v0.33.0-v0.33.7 gates remain inherited.

## Withdrawal Conditions

Withdraw v0.33.8 readiness if CLI argv is passed to shell/subprocess, real providers are called, autonomous runtime is enabled, general tools execute, files are written/edited/patched, trace persistence is added, external trace sinks are used, reference code is executed/imported/installed, secrets are exposed, or any unsafe readiness flag becomes true.

## Handoff

v0.33.8 is a bounded command-surface handoff for v0.33.9 Internal General Agent Runtime MVP Consolidation only.
