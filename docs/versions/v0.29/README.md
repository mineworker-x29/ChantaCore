# ChantaCore v0.29 Restore Documents

This directory is reserved for v0.29.x restore-grade version documents.

v0.29.0 is expected to begin the External Provider Adapter Contract track. A
restore document in this directory must follow
`docs/chanta_core_restore_document_policy.md` and must remain reconstructable:
it should record version scope, restore goal, implemented files, public models,
service behavior, persistence policy, runtime integration, boundaries, OCEL
mapping, PIG/OCPX support, test coverage, restore procedure, limitations, and
future work.

Boundary note: v0.29 restore material must distinguish adapter contract work
from provider invocation, provider registration, network access, credential
storage, command execution expansion, RPA adapter runtime, and external agent
dominion work unless a later reviewed version explicitly implements and verifies
those capabilities.
