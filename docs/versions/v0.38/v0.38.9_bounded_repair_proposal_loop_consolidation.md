## Digestion / Dominion Consolidation

Digestion first remains the policy: read, observe, summarize, model, adapt, and internalize static patterns safely. Dominion remains blocked unless a future explicit gate, strict boundary, audit trace, and human approval exist.

v0.38.9 records digestion_first_policy_applied, dominion_runtime_blocked, external_agent_execution_blocked, infinite_agent_loop_blocked, recursive_self_invocation_blocked, automatic_repair_loop_blocked, retry_loop_blocked, multi_cycle_loop_blocked, repair_execution_blocked, patch_application_blocked, model_provider_invocation_blocked, tool_execution_blocked, bounded_repair_proposal_loop_v1_only, no_independent_autonomous_agent_runtime, mandatory_human_review_before_any_apply, and v039_handoff_future_gated.

External-agent-control patterns remain blocked/future-gated metadata only. OpenCode, Hermes, OpenClaw, Claude Code, and Codex CLI execution remain prohibited.
