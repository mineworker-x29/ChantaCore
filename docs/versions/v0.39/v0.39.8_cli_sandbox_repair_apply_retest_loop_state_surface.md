## Definitions

CLI loop-state surface is a preview-only, in-memory metadata surface over v0.39 repair loop artifacts. It is not CLI execution authority.

Preview-only command is a command specification that classifies and renders metadata. It is not a shell command, subprocess command, arbitrary command, sandbox apply execution, re-test execution, prompt submission, model invocation, or subagent invocation.

CLI repair-loop bundle view is a bounded in-memory view of approval, workspace, sandbox apply, re-test, outcome comparison, process-state, self-prompt draft, human handoff, and blocked-action metadata.

CLI human handoff packet is bounded metadata for human review. It is not human approval capture and does not send anything externally.

Blocked-action surface is an in-memory view of denied runtime actions and safe preview alternatives.

Future v0.39.9 consolidation input is design-stage metadata. It is not v0.39.9 consolidation itself.

It requires preview-only, bounded output, no file write, no external send, and human handoff. It blocks runtime execution, apply, re-test, prompt submission, model invocation, subagent invocation, external agent, auto-continue, retry, multi-cycle, shell, subprocess, arbitrary command, dependency install, network, file export, external send, trace persistence, OCEL write, OCPX persistence, PIG execution, and Dominion.

Rendered view is bounded, redacted, in-memory only, not written to file, and not sent externally.

Handoff packet requires human action and keeps auto-continue and sent-externally false.

Surface result may mark preview rendered or command denied. It keeps all runtime/action/export/persistence flags false.

Surface report is preview-only, may be ready for v0.39.9 consolidation input, and keeps all runtime/action/persistence flags false.

Validation report confirms CLI preview-only behavior and no apply execution, re-test execution, prompt submission, model invocation, subagent invocation, external agent, shell, subprocess, arbitrary command, file export, external send, trace persistence, PIG execution, Dominion, or production certification.

Preview-only guarantee keeps all required no_* flags true.

## Acceptance Criteria
