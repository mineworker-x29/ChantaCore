## Objective

v0.28.6 defines a public-safe runtime profile and deterministic smoke demo flow for Public Alpha candidacy. It creates feature flags, capability matrices, operator/report surfaces, smoke scenario plans, synthetic demo policies, OCEL/OCPX/PIG/Workbench/Memory/Continuity/Safety/CLI demo records, smoke run reports, and a v0.28.7 handoff packet.

It does not implement production runtime, public alpha release, package publishing, release tags, Schumpeter private runtime, adapters, provider invocation, command execution expansion, network calls, runtime continuity injection, autonomous memory execution, or private/company/raw/credential data use.

## Relationship to v0.28.5

v0.28.5 produced Schumpeter preparation handoff boundaries. v0.28.6 consumes those refs and keeps Schumpeter private overlay out of the public alpha runtime profile. Schumpeter handoff is not Schumpeter runtime dependency.

## Relationship to v0.28.7

v0.28.7 may document only the public-safe runtime profile, smoke scenarios, demo flows, and operator/report surfaces from v0.28.6. It must not turn preview-only or future-track items into runtime behavior.

## Core Principles

- Public Alpha runtime profile is not production runtime.
- Smoke demo flow is not production workflow.
- Smoke demo execution is not provider invocation.
- CLI smoke is not command execution expansion.
- Synthetic demo data is not actual user/company data.
- Workbench demo is not UI productionization.
- Memory demo is not autonomous memory-driven execution.
- Continuity preview demo is not runtime continuity injection.
- Schumpeter handoff is not Schumpeter runtime dependency.
- No external adapter, no provider invocation, no command expansion, no private data.

## Feature Flag Matrix

`PublicAlphaFeatureFlagMatrix` separates enabled, preview-only, disabled, and future-track alpha features. Enabled features include OCEL store validation, OCPX projection demo, PIG report demo, Workbench report surfaces, Memory foundation report surfaces, CLI report surfaces, synthetic demo data loading, and safety boundary demo.

Preview-only features include continuity context pack preview, continuity injection bundle preview, durable memory registry dry-run preview, Schumpeter private overlay manifest preview, and external adapter preflight preview.

Disabled or future-track features include provider invocation, command execution expansion, runtime continuity injection, autonomous memory-driven execution, external provider adapter, RPA adapter, A360 adapter, Brity adapter, UiPath adapter, Schumpeter company runtime, and production runtime.

## Runtime Capability Matrix

`AlphaRuntimeCapabilityMatrix` records whether each alpha capability is public-safe, deterministic, synthetic-data-only, provider-free, command-free, file-mutation-free, private-data-free, and independent of Schumpeter private overlay.

## Operator Surface Policy

`AlphaOperatorSurfacePolicy` allows public-safe CLI, report, and Workbench report surfaces. It forbids provider operator surfaces, command execution operator surfaces, and private overlay operator surfaces now. The operator surface is not production UI.

## Smoke Scenario Catalog

`AlphaSmokeScenarioCatalog` includes version check, import smoke, CLI help smoke, synthetic OCEL load, OCPX projection demo, PIG report demo, Workbench report demo, Memory foundation report demo, durable registry dry-run demo, continuity context preview demo, safety boundary demo, and forbidden pattern scan demo. Scenarios are deterministic, synthetic-data-only, private-data-free, provider-free, command-free, mutation-free, and network-free.

## Smoke Input Bundle And Synthetic Data Policy

`AlphaSmokeInputBundle` is synthetic-only and contains no private data, credentials, raw trace, raw transcript, or raw provider output. `AlphaSyntheticDemoDataPolicy` forbids actual user/company data, raw trace/transcript/provider output, and private identifiers; it requires a reproducible synthetic manifest.

## Demo Artifacts

The OCEL demo pack is synthetic-only. The OCPX projection demo and PIG report demo do not invoke providers or execute commands, and PIG is not execution authority. Workbench demo surfaces are report/preview surfaces only and do not mutate files. Memory Foundation demo does not enable autonomous memory execution. Continuity Preview demo is not runtime injection and does not mutate default agent context, DecisionService, or SkillRouter. Safety and CLI demos keep forbidden behavior as explicit non-goals.

## Smoke Run Plan And Report

`AlphaSmokeRunPlan` uses metadata-only mode for this version. Provider invocation, command execution expansion, network calls, and file mutation are not allowed. `AlphaSmokeRunReport` records scenario refs and metadata-only status without provider invocation, command execution, network calls, file mutation, private data use, raw trace use, raw transcript use, or raw provider output use.

## Operator Handoff Packet

`AlphaOperatorHandoffPacket` targets v0.28.7 Alpha Documentation / Onboarding / Example Pack. It includes public-safe demo refs, smoke scenario refs, docs-ready refs, and example-pack candidate refs. It is refs-only and performs no implementation.

Not implemented now: production runtime, package publish, release tag creation, external provider adapter, RPA adapter, A360 adapter, Brity adapter, UiPath adapter, runtime continuity injection, autonomous memory-driven execution, Schumpeter private overlay runtime, and company deployment.
