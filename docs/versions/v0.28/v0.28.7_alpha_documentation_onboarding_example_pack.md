# v0.28.7 Alpha Documentation / Onboarding / Example Pack

Korean name: Alpha 문서·온보딩·예제팩

Track: Public Alpha / Schumpeter Split Preparation

Previous: v0.28.6 Public Alpha Runtime Profile / Smoke Demo Flow

Next: v0.28.8 Alpha Readiness Validation / External Adapter Preflight Gate

## Objective

v0.28.7 creates public-safe documentation, onboarding, and example-pack artifacts from the public-safe runtime profile and smoke demo surfaces defined in v0.28.6. This version is documentation/onboarding/example-pack only.

Documentation is not release. Onboarding is not production support. Example pack is not production dataset. Synthetic example is not actual user/company data. Quickstart is not provider invocation. CLI guide is not command execution expansion. Smoke demo documentation is not production workflow. Schumpeter preparation documentation is not Schumpeter runtime implementation. External adapter future-track note is not external adapter implementation. Future-track documentation must be clearly labeled.

## Relationship

v0.28.6 produced the runtime profile, smoke scenarios, demo surfaces, and operator handoff. v0.28.7 consumes those artifacts and documents only the public-safe enabled, preview-only, disabled, and future-track surfaces.

v0.28.8 must validate docs, examples, links, smoke instructions, safety boundaries, and external adapter preflight gates. v0.28.7 does not prove public release readiness.

## Document Set

The public-safe document set is README, QUICKSTART, ARCHITECTURE, SAFETY_BOUNDARIES, PUBLIC_PRIVATE_BOUNDARY, WORKBENCH_FOUNDATION, MEMORY_FOUNDATION, PUBLIC_ALPHA_RUNTIME_PROFILE, SMOKE_DEMO_FLOW, CLI_REFERENCE, EXAMPLES, SCHUMPETER_PREPARATION, EXTERNAL_ADAPTER_FUTURE_TRACK, and ONBOARDING_CHECKLIST.

## Guides

The quickstart is limited to setup, version/profile inspection, and help/report/smoke review surfaces. The architecture guide describes OCEL-native core, OCPX, PIG, Workbench, Memory foundation, Public Alpha profile, safety boundary, and future external adapter boundary. Runtime and smoke guides distinguish enabled, preview-only, disabled, and future-track capabilities.

## Example Pack

The example pack is synthetic and deterministic. It contains no actual user data, company data, private identifiers, credentials, raw runtime traces, raw transcripts, or raw provider outputs. The example pack manifest and data manifest are documentation artifacts, not production datasets.

## Safety

v0.28.7 forbids public alpha release implementation, package publishing, release tag creation, production runtime claims, provider invocation claims, command execution expansion claims, external adapter enabled claims, Schumpeter runtime claims, private material exposure, actual user/company examples, network calls, runtime continuity injection, references runtime dependency, references code copy, PIG execution authority, and LLM judge as sole documentation authority.
