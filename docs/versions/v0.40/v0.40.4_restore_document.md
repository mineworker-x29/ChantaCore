## 11. Standalone Default Personal Runtime Status

Standalone Default Personal runtime remains closed. ChatService, Orchestrator,
AgentLoop, SkillLoader, SkillExecutor, ProfileRuntime, PromptAssembly,
UserFacingCLI, and event/trace emission are not opened by v0.40.4.

## 12. Why v0.40.4 Is Still Not Standalone

v0.40.4 hardens checkpoint approval metadata. It does not connect user input,
profile loading, prompt assembly, AgentLoop, read-only skill execution, response
generation, or event emission end to end.

## 13. Human Checkpoint Hardening Summary

Human approval is scoped, artifact-bound, freshness-bound, revocable, and
expirable. It is not broad authority.

## 14. Scope-Bound Approval Contract Summary

Only `approve_scoped_rehearsal` for `manual_second_iteration_rehearsal` can
construct a manual second-iteration candidate. Runtime execution remains false.
