# v0.30.8 Limited External Dominion Preview Gate

Parent track: External Dominion Control Plane Foundation

## Relationship To v0.30.0 Through v0.30.7

v0.30.0 defines the External Dominion contract and keeps v0.30 grantability capped at `D3_SIMULATE`.

v0.30.1 defines External Target Inventory, Identity, Availability, and Trust Boundary. Inventory is not permission.

v0.30.2 defines capability observation. Observation is not certification, permission, or runtime readiness.

v0.30.3 defines digestion candidates and assimilation decisions. A digestion candidate is not an active internal artifact.

v0.30.4 defines Dominion authority request, scope, evaluation, decision, and review requirement. Authority decision is not execution.

v0.30.5 defines external delegation candidates and dry-run plans. Delegation candidate is not delegation.

v0.30.6 defines approval, audit, result-boundary, rollback-plan, no-op, failure-classification, retry-deferral, and certification handoff contracts. Approval for certification review is not execution approval.

v0.30.7 defines certification/governance matrix artifacts and v0.30.8 preview gate handoff. Certification pass is not execution approval and is not production certification.

v0.30.8 consumes v0.30.7 gate-review handoff artifacts and decides whether a target/candidate can proceed to v0.30.9 consolidation review only.

## Limited External Dominion Preview Gate

The limited external dominion preview gate is a contract review layer. It checks whether certification handoff artifacts are enough to create a v0.30.9 consolidation handoff.

Preview gate eligibility is not preview execution. Gate pass is not runtime permission.

## Preview Gate Review Versus Preview Execution

Preview gate review examines contract artifacts: certification matrix refs, certification report refs, unresolved requirements, limitations, and runtime-readiness false flags.

Preview execution would contact or operate external systems. v0.30.8 does not do that.

## Status Definitions

Statuses include `unknown`, `not_started`, `eligible_for_gate_review`, `in_gate_review`, `gate_passed_for_consolidation`, `passed_with_limitations_for_consolidation`, `denied`, `deferred`, `blocked`, `future_track`, and `no_op`.

`gate_passed_for_consolidation` is not preview execution. `future_track` is not readiness. No-op is valid.

## Decision Definitions

Decisions include `pass_for_v0309_consolidation`, `pass_with_limitations_for_v0309_consolidation`, `require_more_evidence`, `require_more_boundary_work`, `require_more_certification_work`, `deny`, `defer`, `block`, `future_track`, and `no_op`.

Pass decisions approve v0.30.9 consolidation only. They are not execution approval. Pass with limitations requires limitations.

## Preview Scope

Preview scopes include observation preview, digestion preview, Dominion authority preview, delegation packet preview, dry-run plan/report preview, approval boundary preview, audit boundary preview, result boundary preview, rollback/no-op preview, certification matrix preview, and handoff preview.

Preview scope is not runtime scope. It must prohibit external execution, network access, credential access, command execution, provider invocation, browser automation, RPA control, gateway control, packet send, and rollback execution.

## Deny / Defer Reasons

Reasons include insufficient certification, unresolved blocking cases, missing approval boundary, missing audit policy, missing result boundary, missing rollback/no-op, missing OCEL visibility, missing provider gate inheritance, execution-readiness flag violations, limited preview execution request, production certification request, unresolved network/credential/command/provider/browser/RPA/gateway/delegation surfaces, raw output persistence risk, secret logging risk, memory contamination risk, future gate required, policy block, and unknown.

## Eligibility Matrix

`LimitedExternalDominionPreviewEligibilityMatrix` records required, passed, failed, and unresolved gate conditions plus deny/defer reasons.

It may set `ready_for_v0309_consolidation=True` only when gate review conditions pass. It must keep `ready_for_limited_preview_execution`, `ready_for_execution`, `production_certified`, and `live_adapter_certified` false.

## Preview Candidate

`LimitedExternalDominionPreviewCandidate` records a candidate for preview gate review. Preview candidate is not runtime candidate. `requested_execution` and `ready_for_execution` remain false.

## Preview Audit / OCEL Plan

`LimitedExternalDominionPreviewAuditOCELPlan` describes planned audit artifacts and OCEL visibility requirements.

Preview audit/OCEL plan is not event emission. It does not write events and does not perform audit execution.

## Preview No-Op Plan

`LimitedExternalDominionPreviewNoOpPlan` records a valid no-action route. No-op is valid, is not failure, and executes nothing.

## Preview Gate Report

`LimitedExternalDominionPreviewGateReport` summarizes eligibility, decision, limitations, deny/defer reasons, and followups.

Preview gate report is not production certification and is not preview execution.

## v0.30.9 Consolidation Handoff

`ExternalDominionV0309ConsolidationHandoff` carries the preview gate report to v0.30.9 External Dominion Foundation Consolidation.

v0.30.9 handoff is not runtime readiness. `ready_for_v0309_consolidation` may be true only for consolidation and not execution.

## Safe Defaults

- Preview gate eligibility is not preview execution.
- Preview candidate is not runtime candidate.
- Preview scope is not runtime scope.
- Preview decision is not execution approval.
- Preview audit/OCEL plan is not event emission.
- Preview no-op is valid.
- Preview gate report is not production certification.
- v0.30.9 handoff is not runtime readiness.
- `ready_for_limited_preview_execution` remains false.
- `ready_for_execution` remains false.
- `approved_for_limited_preview_execution` remains false.
- `approved_for_execution` remains false.
- D4-D9 remain future-track.

## Flow

`V0308PreviewGateHandoff -> PreviewEligibilityMatrix -> PreviewScope -> PreviewCandidate -> PreviewDecision -> PreviewAuditOCELPlan or PreviewNoOpPlan -> PreviewGateReport -> V0309ConsolidationHandoff`

The flow creates contract/governance artifacts only.

## Hard Prohibitions

v0.30.8 must not:

- execute external targets;
- execute preview;
- approve real execution;
- certify production readiness;
- certify live adapter readiness;
- execute rollback;
- retry;
- invoke external agent harnesses;
- transfer delegation packets;
- spawn processes;
- call provider SDKs;
- perform network calls;
- access credentials or secrets;
- use subprocess, shell, or command execution;
- implement RPA runtime control;
- implement browser automation;
- implement gateway/channel message control;
- persist raw external output as memory;
- mark any target as live-runtime-ready;
- bypass v0.29 provider adapter gates;
- raise v0.30 grantability beyond `D3_SIMULATE`;
- grant `D7_EXECUTE_NETWORK_PREVIEW`, `D8_DELEGATE_AGENT`, or `D9_GATEWAY_CONTROL`;
- set `ready_for_execution=True`;
- set `ready_for_limited_preview_execution=True`;
- set `limited_preview_execution_ready_now=True`;
- set `production_certified=True`;
- set `live_adapter_certified=True`;
- treat preview gate pass as preview execution approval;
- treat gate review as runtime permission.

## Gate Inheritance

- `live_adapter_runtime_ready = False`

- `credential_access_ready = False`
- `command_execution_ready = False`
- `rpa_runtime_control_ready = False`
- `gateway_control_ready = False`

No runtime readiness flag changes in v0.30.8.

## Withdrawal Conditions

Withdraw this release interpretation if:

- any helper executes, fetches, invokes, registers, writes external data, accesses credentials, or contacts an external runtime;
- any helper approves execution or limited preview execution;
- any helper emits OCEL events;
- any helper executes rollback or retry;
- any preview gate pass is treated as runtime permission;
- `ready_for_execution`, `ready_for_limited_preview_execution`, `approved_for_execution`, `approved_for_limited_preview_execution`, `production_certified`, or `live_adapter_certified` becomes true;
- D7, D8, or D9 are granted;
- runtime readiness flags change.

## Handoff To v0.30.9

v0.30.8 is handoff-ready for v0.30.9 External Dominion Foundation Consolidation only. It does not enable limited preview execution, runtime readiness, external execution, provider invocation, network access, credential use, command execution, RPA/browser/gateway runtime, rollback, retry, or packet transfer.
