It creates returned/in-memory OCEL-style trace packet metadata for the human-approved sandbox patch apply lifecycle.

## Definition

A Patch Apply Sandbox OCEL Trace Packet is bounded metadata describing the sandbox apply lifecycle. It may include trace events, trace objects, trace relations, trace attributes, sandbox apply lifecycle records, bounded agentic task lifecycle records, Digestion/Dominion boundary records, emission decisions, validation reports, and readiness reports.

The trace packet is not persistent trace write. An OCEL-style packet is not an OCEL file write. Trace event/object/relation creation is not runtime execution, registry mutation, or operational control. Sandbox apply trace is not live apply permission. Agentic task trace is not autonomous runtime. Digestion/Dominion trace is not Dominion runtime.

## Policy

PatchApplyTracePolicy allows returned/in-memory trace packet creation only. It blocks raw diff, raw source, raw sandbox file content, raw validation report payloads, secrets, credentials, tokens, full file content, persistent writes, external sinks, OCEL file writes, JSONL writes, log writes, database writes, sandbox file writes, live workspace writes, patch application, test execution, shell execution, external agent execution, and Dominion runtime.

Trace attributes are sanitized, redacted, and bounded. Secret/key/token/credential/pem/id_rsa-like payloads are prohibited. If artifacts are insufficient, the layer returns trace gaps or blocked validation findings rather than expanding behavior.

## Artifacts

PatchApplyTraceObject, PatchApplyTraceEvent, PatchApplyTraceRelation, and PatchApplyTraceAttribute are metadata only. They do not execute traced events and do not mutate registries.

SandboxApplyLifecycleTraceRecord records candidate, approval, dry-run, sandbox manifest, sandbox apply, and post-apply validation references. live_write_performed, patch_application_performed, and production_certified remain false.

AgenticTaskLifecycleTraceRecord records run packet, result, stop reason, and step record refs. single_cycle_only and human_handoff_required are true. automatic_retry_allowed, automatic_repair_allowed, independent_agent_runtime, and multi_cycle_loop remain false.

DigestionDominionApplyTraceRecord records digestion_first_policy_applied, dominion_runtime_blocked, external_agent_execution_blocked, infinite_agent_loop_blocked, recursive_self_invocation_blocked, automatic_repair_loop_blocked, bounded_agentic_task_only, no independent autonomous agent runtime, and mandatory human handoff after cycle as blocked/future-gated metadata only.

PatchApplySandboxTracePacket is returned/in-memory metadata, not persistent storage. It may reference sandbox apply success but does not grant live apply permission. It may reference bounded agentic operation success but does not grant autonomous runtime.

PatchApplyTraceEmissionInput, PatchApplyTraceEmissionDecision, PatchApplyTraceEmissionReport, PatchApplyTraceEmitter, PatchApplyTraceValidationReport, PatchApplyTraceNoPersistenceGuarantee, and V0367ReadinessReport preserve the same boundary.
