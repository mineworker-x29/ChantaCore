# Quickstart

This quickstart is for v0.28.7 public-safe documentation and smoke review only. Quickstart is not provider invocation. CLI guide is not command execution expansion.

## Steps

1. Inspect the package version and local metadata.
2. Review `python -m chantacore.cli alpha runtime report`.
3. Review `python -m chantacore.cli alpha smoke report`.
4. Review `python -m chantacore.cli alpha docs report`.
5. Inspect synthetic examples in `docs/EXAMPLES.md`.

Forbidden steps: provider invocation, command execution expansion, package publish, release tag creation, Schumpeter private runtime, and external adapter run.
