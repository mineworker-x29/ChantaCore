# Safety Boundaries

v0.28.7 documentation must preserve these boundaries:

- no provider invocation
- no command execution expansion
- no network calls
- no runtime continuity injection
- no autonomous memory execution
- no private data
- no credentials or secrets
- no raw traces
- no raw transcripts
- no raw provider outputs
- no external adapter implementation
- no Schumpeter private runtime

Documentation is not release, and onboarding is not production support.
