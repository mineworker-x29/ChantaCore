# Architecture

This architecture note describes public-safe ChantaCore documentation surfaces for v0.28.7. It does not claim production runtime readiness.

## Public Surfaces

OCEL-native core records object/event/effect visibility. OCPX projects read-model state. PIG reports guidance and must not become execution authority. Workbench and Memory foundation surfaces are report/preview oriented in the public alpha preparation track.

## Future Boundary

External adapters, RPA adapters, A360, Brity, UiPath, and provider invocation remain future-track and require v0.28.8 preflight plus later v0.29 contract work before implementation.
