# Smoke Demo Flow

Smoke demo flow is not production workflow. Smoke demo execution is not provider invocation.

Documented scenarios are version check, import smoke, CLI help smoke, synthetic OCEL load, OCPX projection demo, PIG report demo, Workbench report demo, Memory foundation report demo, durable registry dry-run demo, continuity context preview demo, safety boundary demo, and forbidden pattern scan demo.

All scenarios are deterministic, synthetic/public-safe, and require no provider, command expansion, network call, file mutation, private data, raw trace, transcript, or provider output.
