# Public Alpha Runtime Profile

The public alpha runtime profile is not production runtime. It documents enabled, preview-only, disabled, and future-track surfaces from v0.28.6.

Enabled surfaces include OCEL store validation, OCPX projection demo, PIG report demo, Workbench report surfaces, Memory foundation report surfaces, CLI report surfaces, synthetic demo data loading, and safety boundary demo.

Preview-only surfaces include continuity context preview, continuity injection bundle preview, durable registry dry-run preview, Schumpeter private overlay manifest preview, and external adapter preflight preview.

Disabled or future-track surfaces include provider invocation, command execution expansion, runtime continuity injection, autonomous memory execution, external provider adapters, RPA adapters, A360, Brity, UiPath, Schumpeter company runtime, and production runtime.
